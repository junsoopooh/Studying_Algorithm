3. 힌트 확인 후 다시 품
```
def solution(begin, end):
    result = []
    for target in range(begin,end+1):
        tmp = []
        for block in range(1,target):
            if target % block == 0:
                tmp.append(block)
        if tmp:
            result.append(max(tmp))
        else:
            result.append(0)
            
    return result

```
