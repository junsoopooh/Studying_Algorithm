## TMI
### 2 x n 타일링
- 값을 저장해 놓는 dp 배열은 인덱스가 0부터 시작하기 때문에 길이가 [0] * n이 아닌 [0] * (n+1)로 해야만 한다. 그래야 우리가 찾고자 하는 dp[n]의 값을 구할 수 있다.
